문제

초를 입력 받아, 시, 분, 초를 계산하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
초를 입력하세요 : 3661
3661초는 1시간 1분 1초입니다.


초를 입력하세요 : 61
61초는 0시간 1분 1초입니다.


초를 입력하세요 : 0
0초는 0시간 0분 0초입니다.

알고리즘

프로그램 시작 
   전체 초를 입력받기 
   시, 분, 초를 계산 
      시를 계산 
      분을 계산 
      초를 계산 
   시, 분, 초를 출력 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter14/TimeMain.c

	#include <stdio.h>

	 

	// 프로그램 시작 

1	int main() {

		int totalSeconds = 0, hours = 0, minutes = 0, seconds = 0;

	 

		// 전체 초를 입력받기 

		printf( "초를 입력하세요 : " );

2		scanf( "%d", &totalSeconds );

	 

		// 시, 분, 초를 계산 

		// 시를 계산 

3		hours  = totalSeconds / ( 60 * 60 );

		// 분을 계산 

4		minutes = ( totalSeconds % ( 60 * 60 ) ) / 60;

		// 초를 계산 

5		seconds = totalSeconds % 60;

	 

		// 시, 분, 초를 출력 

6		printf( "%d초는 %d시간 %d분 %d초입니다.", totalSeconds, hours, minutes, seconds );

	 

		// 프로그램 종료 

7		return 0;

	}
