문제

초를 입력 받아, 시, 분, 초를 계산하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
초를 입력하세요 : 3661
3661초는 1시간 1분 1초입니다


초를 입력하세요 : 61
61초는 0시간 1분 1초입니다


초를 입력하세요 : 0
0초는 0시간 0분 0초입니다

알고리즘

 프로그램 시작 
    전체 초를 입력받기 
    시, 분, 초를 계산 
    시, 분, 초를 출력 
 프로그램 종료 
 
 전체 초를 입력받는 함수 
    사용자에게 전체 초를 입력받기 
 
 시, 분, 초를 계산하는 함수 
    시를 계산 
    분을 계산 
    초를 계산

F2b	Time calculateTime( int totalSeconds ) {

		Time time = { 0, 0, 0 };

		// 시를 계산 

		time.hours = totalSeconds / ( 60 * 60 );

		// 분을 계산

		time.minutes = totalSeconds % ( 60 * 60 ) / 60;

		time.seconds = totalSeconds % 60;

	}
